# 🌦️ Weather App

A simple and responsive Weather App built using **HTML**, **CSS**, and **JavaScript**. It fetches real-time weather data from the **WeatherAPI** and displays current weather information for any city entered by the user.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather condition
- 💧 Displays humidity
- 🌬️ Shows wind speed
- 🖼️ Weather condition icon
- 📱 Responsive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Fetch API)
- WeatherAPI

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

4. Enter any city name and click **Search**.

## 🔑 API Used

This project uses the **WeatherAPI**.

API Endpoint:

```
https://api.weatherapi.com/v1/current.json
```

Official Website:

https://www.weatherapi.com/
